Mobile Store : متجري الإلكتروتي

وصف المشروع

موقع تجاري مصغر يتيح للمستخدمين تصفح المنتجات،وإتمام عملية الشراء. 
يدعم الموقع إدارة المنتجات والطلبات وبيانات العملاء

أهداف المشروع

1-تسهيل تجربة التسوق الإلكتروني للمستخدمين من خلال واجهة بسيطة وسهلة الاستخدام

2-تنظيم وتطوير عملية إدارة المتاجر الصغيرة عبر توفير لوحة تحكم لإدارة المنتجات والمخزون

3-توفير وسيلة فعالة للمدير لمتابعة الطلبات وحركة المبيعات بشكل دقيق

فكرة المشروع
فكرة المشروع تقوم على توفير منصة بسيطة للتسوق الإلكتروني تناسب المتاجر الصغيرة، 
مع لوحة تحكم للمدير لإدارة المنتجات والمخزون ومتابعة الطلبات

المميزات:
عرض المنتجات مع الصور والأسعار
لوحة تحكم للمدير لمتابعة المبيعات
