 متجر الهواتف الذكية (Mobile Store)
 
  وصف المشروع
موقع تجاري لإدارة وبيع الهواتف الذكية يتيح للمستخدمين تصفح الهواتف المتوفرة، أسعارها، ومواصفاتها. يدعم الموقع إدارة المنتجات وبيانات العملاء وصلاحيات المستخدمين.

  فكرة المشروع
فكرة المشروع تقوم على توفير منصة بسيطة ومتكاملة لبيع وإدارة الهواتف، مع لوحة تحكم للمدير لإدارة المنتجات من مواصفات وأسعار ومتابعة بيانات العملاء المسجلين.
• عرض الهواتف مع تحديد الشركة والموديل والسعر والسعة
• إدارة بيانات العملاء وإضافتهم 
• لوحة تحكم للمدير لمتابعة المتجر 

  طريقة تشغيله
1. استنساخ المشروع:
git clone https://github.com

2. إعداد قاعدة البيانات:
استورد ملف database.sql إلى phpMyAdmin.

3. تشغيل المشروع:
افتح index.php في المتصفح مباشرة.
